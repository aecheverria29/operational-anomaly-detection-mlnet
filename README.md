\# Operational Anomaly Detection with ML.NET



Proyecto en .NET enfocado en detección de anomalías sobre datos operativos o de negocio, con una interfaz web construida en ASP.NET Core MVC.



\## Objetivo

Demostrar cómo integrar ML.NET en una aplicación .NET para identificar comportamientos inusuales en métricas de negocio, tickets o eventos operativos.



\## Tecnologías base

\- ASP.NET Core MVC

\- .NET 8

\- ML.NET

\- C#

\- Git / GitHub



\## Estado actual

Proyecto base inicial creado con plantilla MVC.
